# Linjalla

**Linjalla** is an iOS app project for viewing bus information, built with Xcode/Swift. ("RT Bus" was the working name during development.)

Repository: https://github.com/Valleyaapo/Linjalla

## Requirements
- macOS + Xcode (latest stable recommended)
- iOS Simulator or a physical iOS device

## Getting started
1. Clone the repo:
   ```bash
   git clone https://github.com/Valleyaapo/Linjalla.git
   cd Linjalla
   ```
2. Open the Xcode project:
   - Open `RT Bus.xcodeproj` in Xcode
3. Select a simulator/device and Run (`⌘R`).

## Project structure
- `RT Bus/` — main iOS app source
- `RT Bus.xcodeproj/` — Xcode project configuration
- `RT BusTests/` — unit tests
- `RT BusUITests/` — UI tests
- `Package.swift` / `Package.resolved` — Swift Package Manager dependencies
- `docs/privacy.md` — privacy policy
- `animation_state_manager.md` — development notes
